Mi Experiencia

¿Como fue mi Experiencia al crear mi Cuenta En Github?

Al principio, la idea me generaba un poco de inseguridad, porque no sabía exactamente qué tan complejo sería el proceso. Había escuchado que GitHub era una de las principales plataformas para compartir proyectos, colaborar en equipo y llevar un control de versiones, así que pensé: “Si quiero crecer como futuro profesional en sistemas, necesito dar este paso.”

¿Que es Github?

Github

Git es una herramienta que controla las versiones de un proyecto y permite trabajar en paralelo sin perder cambios. GitHub es una plataforma en línea que funciona como una nube especializada en proyectos con Git. Permite guardar proyectos en línea y tener un historial de cambios. Ver mas
